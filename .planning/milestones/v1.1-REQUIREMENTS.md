## Terminal Interactivity

- [x] **TERM-INT-01**: Terminal MUST be immediately interactive on page load — no click-to-focus required
  - Outcome: Validated. terminal.focus() called after open() in useEffect.
- [x] **TERM-INT-02**: Terminal MUST auto-focus when switching tabs between sessions
  - Outcome: Validated. Component remount triggers focus via useEffect.
- [x] **TERM-SCROLL-01**: Terminal MUST support scrollback (mouse wheel scroll up through history)
  - Outcome: Validated. tmux mouse mode enabled, scroll enters copy-mode.
- [x] **TERM-SCROLL-02**: Scrollback must show meaningful history, not just the current viewport
  - Outcome: Validated. history-limit set to 50,000 lines.

## Prompt Panel

- [x] **PROMPT-01**: Dropdown MUST reflect the currently selected session
  - Outcome: Validated. derivedAgentId computed from activeInstances lookup.
- [x] **PROMPT-02**: When user selects a tab, prompt dropdown auto-syncs to that session's agent
  - Outcome: Validated. useEffect resets on selectedAgentId change.
- [x] **PROMPT-03**: Dropdown MUST allow manual override to any configured agent
  - Outcome: Validated. Manual override persists until next tab switch.
- [x] **PROMPT-04**: Ctrl+Enter MUST send the prompt
  - Outcome: Validated. Tested via Playwright E2E.
- [x] **PROMPT-05**: Send button MUST work
  - Outcome: Validated. Gateway API delivery confirmed.
- [x] **PROMPT-06**: Prompt goal: send to agent via OpenClaw Gateway
  - Outcome: Validated. /v1/chat/completions endpoint integration working.
