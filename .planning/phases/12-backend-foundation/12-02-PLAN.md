  <action>
Create `src/server/routes/gsdRoutes.ts` following the instanceRoutes.ts Router pattern.

**Imports:**
- `Router`, `Request`, `Response` from `express`
- `spawn` from `child_process`
- `execFile` from `child_process` + `promisify` from `util` → `const execFileAsync = promisify(execFile)`
- `readFile` from `fs/promises`
- `path` from `path`
- `gsdRegistryService` from `../services/GsdRegistryService.js`
- `gsdHookLogWatcher` from `../services/GsdHookLogWatcher.js` (for hooks/log REST endpoint)

**Constants:**
- `SPAWN_SH_PATH = '/home/forge/.openclaw/workspace/skills/gsd-code-skill/scripts/spawn.sh'`
- `MENU_DRIVER_PATH = '/home/forge/.openclaw/workspace/skills/gsd-code-skill/scripts/menu-driver.sh'`
- `ALLOWED_ACTIONS = new Set(['snapshot', 'enter', 'esc', 'clear_then', 'choose', 'submit', 'type'])`
- `SESSION_NAME_RE = /^[a-zA-Z][a-zA-Z0-9_-]*$/`
- `AGENT_NAME_RE = /^[a-zA-Z][a-zA-Z0-9_-]*$/`
- `COMMAND_ARG_RE = /^[\/a-zA-Z0-9 @:._-]+$/`
- `WORKDIR_PREFIX = '/home/forge/'`

**Endpoints (6 total):**

1. `GET /api/gsd/registry` — call `gsdRegistryService.getRegistry()`, return JSON. Wrap in try-catch, 500 on error.

2. `PATCH /api/gsd/registry/agents/:agentId` — validate `agentId` matches AGENT_NAME_RE. Extract `enabled` from request.body. Validate `enabled` is a boolean. Call `gsdRegistryService.patchAgent(agentId, { enabled })`. Return updated agent. 404 if agent not found (catch and check error message). 400 for validation failures.

3. `POST /api/gsd/spawn` — extract `agentName`, `workdir`, `firstCommand` from request.body.
   - Validate agentName: required, string, matches AGENT_NAME_RE → 400
   - Validate workdir: required, string → 400. Then `path.resolve(workdir)` and assert `startsWith(WORKDIR_PREFIX)` → 400 with "workdir must be within /home/forge/"
   - Validate firstCommand (optional): if present, must be string matching COMMAND_ARG_RE → 400
   - Fire-and-forget: `spawn(SPAWN_SH_PATH, [agentName, resolvedWorkdir, ...(firstCommand ? [firstCommand] : [])], { detached: true, stdio: 'ignore' })`, then `child.unref()`
   - Return 202 with `{ message: 'Spawn initiated', agentName, workdir: resolvedWorkdir }`

4. `POST /api/gsd/sessions/:session/command` — validate session matches SESSION_NAME_RE → 400. Extract `action`, `args` from body. Validate action is in ALLOWED_ACTIONS → 400. If args present, validate string matching COMMAND_ARG_RE → 400. Call `execFileAsync(MENU_DRIVER_PATH, [session, action, ...(args ? [args] : [])])`. Return `{ dispatched: true, output: stdout.trim() }`. 500 on execFile error.

5. `GET /api/gsd/sessions/:session/state` — validate session matches SESSION_NAME_RE → 400. Look up agent in registry by `tmux_session_name === session`. 404 if not found. Read `path.join(agentEntry.working_directory, '.planning', 'STATE.md')`. Return `{ sessionName, stateContent }`. If ENOENT, return `{ sessionName, stateContent: null }`. 500 on other errors.

6. `GET /api/gsd/hooks/log` — extract `lines` query param (default 200, max 1000). Call `gsdHookLogWatcher.readLastLines(parsedLines)`. Return `{ lines }`. **NOTE:** This requires making `readLastLines` public on GsdHookLogWatcher — update the method visibility from private to public.

**Export:** `export { router as gsdRoutes };`

**Validation approach:** Inline regex + early return (no middleware library). This follows existing route patterns and the research recommendation. Every validation failure returns 400 with a descriptive error message.
