    Create `src/client/hooks/usePluginRegistry.ts`:

    - Define `STORAGE_KEY = 'warden:plugin-enabled'`
    - Create helper function `loadPersistedState(pluginIds: string[]): Record<string, boolean>` that reads from localStorage with try-catch, defaults new plugins to `true` (enabled)
    - Export `usePluginRegistry()` hook that returns `{ plugins, enabledState, enabledPlugins, togglePlugin }`:
      - `plugins`: the `registeredPlugins` record (all plugins)
      - `enabledState`: `Record<string, boolean>` initialized from localStorage via lazy `useState` initializer
      - `enabledPlugins`: `PluginModule[]` — filtered array of only enabled plugins
      - `togglePlugin`: `(pluginId: string) => void` — flips the boolean in enabledState (use `useCallback`)
    - Persist `enabledState` to localStorage via `useEffect` on every change, with try-catch for quota errors

    Create `src/client/plugins/example-plugin.tsx`:
    - Import `PluginModule` type from `@shared/pluginTypes.js`
    - Define `manifest` with `as const satisfies PluginManifest`:

      - `description: 'Demonstrates the plugin registry system'`

    - Export default `{ manifest, PanelComponent: ExamplePluginPanel } satisfies PluginModule`
