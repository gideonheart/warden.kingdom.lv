Create `src/server/services/ApprovalCallbackHandler.ts`:

```typescript
class ApprovalCallbackHandler {
  constructor(
    private approvalTracker: ApprovalStateTracker,
    private tmuxSessionManager: TmuxSessionManager
  )
  register(bot: Bot): void  // registers bot.callbackQuery(/^approve:(.+)$/, handler)
}
```

The `register(bot)` method attaches a callback query handler that:
1. Extracts `sessionName` from `ctx.match[1]` (regex capture)
2. Reads `WARDEN_TELEGRAM_OPERATOR_ID` from env; if missing, answers with 'Bot misconfigured' and returns
3. Compares `ctx.callbackQuery.from.id` to parsed operator ID; rejects non-operators with 'Not authorized' (show_alert: true)
4. Gets approval record from tracker; if not found or expired (Date.now() - sentAt > APPROVAL_EXPIRY_MS), answers with 'Approval expired' (show_alert: true)
5. Checks `consumed` flag — if true, answers silently and returns (idempotency)
6. Calls `markConsumed(sessionName)` **synchronously** BEFORE the async tmux call (prevents double-tap race)
7. Calls `tmuxSessionManager.sendPromptToSession(sessionName, '1')` wrapped in try/catch
8. If tmux call succeeds: answers with 'Approved!', then calls `bot.api.editMessageText(chatId, messageId, originalText + '\n\n_Approved at HH:MM_', { parse_mode: 'Markdown' })` — no reply_markup = button removed
9. If tmux call fails (session gone): answers with 'Session no longer available' (show_alert: true), does NOT edit message
