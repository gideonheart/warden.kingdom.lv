  <action>
    1. **Database** (`src/server/database/DatabaseConnection.ts`):
       - Add `getLastProjectPaths(): Record<string, string>` method:
         ```sql
         SELECT agent_id, project_path
         FROM instances
         WHERE project_path != ''
         ORDER BY last_active_at DESC
         ```
         Process results to return a map of agentId -> most recent non-empty project_path.
         Since the query returns multiple rows per agent (all sessions), use a Map and only keep the first occurrence (most recent due to ORDER BY).

    2. **API — Last project paths** (`src/server/routes/instanceRoutes.ts`):
       - Add `GET /api/agents/last-projects` endpoint:
         Returns `{ paths: Record<string, string> }` from `database.getLastProjectPaths()`.
         No authentication required (matches existing API pattern).

    3. **API — Extend start endpoint** (`src/server/routes/instanceRoutes.ts`):
       - Modify `POST /api/instances/start` to accept optional `projectPath` in request body:
         ```typescript
         const { agentId, projectPath: overridePath } = request.body as { agentId?: unknown; projectPath?: unknown };
         ```
       - If `overridePath` is provided (non-empty string), use it instead of looking up workspace from openclaw.json:
         - Validate it's a non-empty string
         - Still look up `agentName` from openclaw.json (needed for instance creation)
         - Use the overridePath as `projectPath` and derive `projectSlug` from it
       - If `overridePath` is not provided, behavior remains unchanged (workspace from openclaw.json)
       - This supports LNCH-03: operator can override the pre-filled project path before launching

         const handleQuickLaunch = useCallback(async (agentId: string, projectPath: string) => {
           const response = await fetch('/api/instances/start', {
             method: 'POST',
             headers: { 'Content-Type': 'application/json' },
             body: JSON.stringify({ agentId, projectPath }),
           });
           if (!response.ok) {
             const data = await response.json();
             throw new Error(data.error || 'Launch failed');
