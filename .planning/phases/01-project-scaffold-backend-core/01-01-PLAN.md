<task type="auto">
  <name>Implement PTY cleanup for graceful shutdown</name>
  <files>src/server/services/TerminalStreamService.ts</files>
  <action>
Add `killAllPtyProcesses()` method to TerminalStreamService for graceful shutdown PTY cleanup.

Implementation:
```typescript
async killAllPtyProcesses(): Promise<void> {
  console.log(`[TerminalStream] Killing ${this.activeStreams.size} active PTY processes`);
  for (const [socketId, stream] of this.activeStreams.entries()) {
    stream.ptyProcess.kill();
    this.activeStreams.delete(socketId);
  }
}
```

This prevents PTY zombie processes on server shutdown. Each PTY process must be explicitly killed before exit, or they remain in process table as defunct processes.

Reference: RESEARCH.md Pitfall 1 (PTY Zombie Processes)
