  <action>
    **ControlsTab.tsx** — The spawn and dispatch handlers fire 6 `setTimeout(() => setXStatus(null), N)` calls without cleanup. Fix:
    1. Add `import { useRef, useEffect }` (useEffect for cleanup, useRef for timer tracking).
    2. Add `const spawnTimerRef = useRef<ReturnType<typeof setTimeout> | null>(null)` and `const dispatchTimerRef = useRef<ReturnType<typeof setTimeout> | null>(null)`.
    3. In handleSpawn, replace all 3 bare `setTimeout(() => setSpawnStatus(null), 5000)` with: `spawnTimerRef.current = setTimeout(() => setSpawnStatus(null), 5000)`.
    4. In handleDispatch, replace all 3 bare `setTimeout(() => setDispatchStatus(null), N)` with: `dispatchTimerRef.current = setTimeout(() => setDispatchStatus(null), N)`.
    5. Add a useEffect cleanup: `useEffect(() => { return () => { if (spawnTimerRef.current) clearTimeout(spawnTimerRef.current); if (dispatchTimerRef.current) clearTimeout(dispatchTimerRef.current); }; }, []);`

    **gsdShared.tsx CopyButton** — The clipboard `.then()` fires `setTimeout(() => setCopied(false), 2000)` without cleanup. Fix:
    1. Add `import { useRef, useEffect }` to the import line.
    2. Add `const timerRef = useRef<ReturnType<typeof setTimeout> | null>(null)` inside CopyButton.
    3. Replace `setTimeout(() => setCopied(false), 2000)` with `timerRef.current = setTimeout(() => setCopied(false), 2000)`.
    4. Add `useEffect(() => { return () => { if (timerRef.current) clearTimeout(timerRef.current); }; }, []);`
  </action>
